addInputAnalog()

Description

Adds a control in the board. This control is a slider that the user can change to set the value of a parameter. When the slider changes, a callback function is called with the new value as a parameter.

Syntax

webgui.addInputAnalog("Name of the control", minvalue, maxvalue, defaultvalue, fnSetValue)

Headers

int addInputAnalog(const char * name, float minvalue, float maxvalue, float defaultvalue, CallbackTypeFloat fnAction);

Parameters

name: The name of the slider

minvalue: Is the minimum value that slider can generate. This means that when the user slides the slider completely at left, minvalue will be the argument of fnAction.

maxvalue: Is the maximum value that slider can generate. This means that when the user slides the slider completely at right, maxvalue will be the argument of fnAction. This value must be grater than minvalue.

defaultvalue : This value represents the default position of the slider. This value must be equal or grater than minvalue and equal or lower than maxvalue.

fnAction: Pointer to function that takes a float as a parameter and returns void. This function will be called with a float as an argument that represents the position of the slider. The value of this float will be between minvalue and maxvalue, included. It is called each time the slider is changed so when the user slides the position of slider, probably multiple calls to this function will occur, one each time slider moves one step.

Returns

The int id of the element

Example

#include <Webgui.h> 
#include <WiFi.h> 
  
Webgui webgui; //initialize an instance of the class 

char ssid[] = "Mywifi"; //  your network SSID (name)  
char pass[] = "0123ABCDEF";    // your network password (use for WPA, or use as key for WEP) 
int keyIndex = 0;// your network key Index number (needed only for WEP) 
char server[] = "192.168.1.17"; 
WiFiClient client; 
int status = WL_IDLE_STATUS; 

float power = 0.1; // 0 - 1 
int pin = 10; 

void actionFloat(float newpower){ 
    power = newpower; 
    Serial.print("Led set to power: "); 
    Serial.print((long)(100.0*power)); 
    Serial.println(" %"); 
} 

void setup(){ 
    pinMode(pin, OUTPUT);// sets the digital pin as output 
    Serial.begin(9600); 
    while(!Serial); 
    Serial.println("Serial ready @ 9600 bauds"); 
    // attempt to connect to Wifi network: 
    while ( status != WL_CONNECTED) { 
        status = WiFi.begin(ssid, keyIndex, pass); 
    } 
    Serial.println("Connected to wifi !"); 
    while(!webgui.connect(&client, server)); //try to connect as many times as needed
    Serial.println("Connected to webgui server !"); 
    webgui.reset(); 
    webgui.addInputAnalog("Power of the LED signal", 0.0, 0.25, power, &actionFloat); 
} 

void loop(){ 
    webgui.update(); 
    int power255 = (int)(power*255.0);  
    analogWrite(pin, power255); //duty cicle must be in 0 - 255 range
    delay(250); 
}
