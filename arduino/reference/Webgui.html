Webgui Library

This library allows you to connect to Webgui server in order to communicate with the graphical user interface web that webgui server serves.

To use this library you need the WiFi Shield module to connect to the webgui server.

You need to create an instance of the class Webgui to use it. Go to update function reference to see an entire aplication example that uses Webgui library.
